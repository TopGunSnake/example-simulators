Module fo_sim::state_machine[−][src]

Expand description

Provides the functions and enums for maintaining the FO state machine.

The FO State Machine uses the top level FoState for representing the state of the FO. The workhorse of this module, and the intended component for use is the state_machine_loop, which provides an async function for use in a runtime.
